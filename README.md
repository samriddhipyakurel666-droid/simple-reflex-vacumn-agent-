

## 📖 Project Description
Beginner Python project implementing a Simple Reflex Vacuum Agent in Google Colab.  
The agent operates in a two‑location environment (A/B), randomly initialized as clean or dirty.  
It starts in a random location, senses the state, cleans if dirty, moves to the other location, and repeats.  
The program prints the initial environment, each action, the final state, and a performance score.

--

##  Features
- Two‑location environment (A and B)
- Random initialization of clean/dirty states
- Random agent starting position
- Simple `if‑else` logic for decisions
- Clear print outputs for each step
- Performance score based on cleaned locations

---

##  Requirements
- Python 3.x
- Google Colab or any Python IDE
